# Uitvoeren op de nieuwe server

## Optie A (als de scripts al op de server staan)

Ga naar de map van deze repository en run:

sudo ./install_server.sh

## Optie B (copy/paste direct in de SSH-terminal op de nieuwe server)

Log eerst in via SSH als `root` en plak daarna deze command:

curl -fsSL https://raw.githubusercontent.com/michaeldbr/linux_server_install/main/scripts/00_common/remote_bootstrap.sh | REPO_URL='https://github.com/michaeldbr/linux_server_install.git' BRANCH='main' bash

Wat dit doet op de nieuwe server:
- Installeert tijdelijk `git` (alleen als nodig)
- Haalt deze repository binnen
- Voert `install_server.sh` uit
- Installeert Webmin, vraagt wachtwoord voor Webmin-user `michael`, en zet Webmin op poort `40112`
- Verwijdert tijdelijke scriptbestanden weer
- Verwijdert `git` weer als die alleen voor deze run is geïnstalleerd
